ALTER TABLE

ALTER TABLE lets you modify an existing table’s structure.

Here’s what you can do:

1 Add a new column


ALTER TABLE table_name
ADD column_name datatype;

Examples:

Add a birthdate column:


ALTER TABLE employees
ADD birthdate DATE;

Add a nullable text column:


ALTER TABLE employees
ADD nickname VARCHAR(50);

Add multiple columns at once:


ALTER TABLE employees
ADD (
    department VARCHAR(100),
    hire_date DATE
);

2 Modify an existing column

Different databases use different wording, but here are the common forms:

Change data type:


ALTER TABLE employees
MODIFY salary DECIMAL(10,2);

Rename a column:

MySQL:


ALTER TABLE employees
CHANGE COLUMN old_name new_name VARCHAR(100);

3 Drop a column (delete it)


ALTER TABLE employees
DROP COLUMN nickname;

Drop multiple columns:


ALTER TABLE employees
DROP COLUMN department,
DROP COLUMN hire_date;

Deleting columns affects data permanently, so double-check before firing that query.

Extra power moves

Add a column with a default:


ALTER TABLE employees
ADD status VARCHAR(20) DEFAULT 'active';

Set a column to NOT NULL later:


ALTER TABLE employees
MODIFY status VARCHAR(20) NOT NULL;

Add a primary key:


ALTER TABLE employees
ADD PRIMARY KEY (id);

Drop a primary key:


ALTER TABLE employees
DROP PRIMARY KEY;

Quick practice

Create this table:


CREATE TABLE employees (
    id INT PRIMARY KEY,
    name VARCHAR(100)
);

Now apply changes:

  1. Add salary as a decimal value
  2. Rename name to full_name
  3. Remove salary
