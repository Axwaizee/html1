Lists practice

Lists help organize content. There are two main types: ordered and unordered.

Unordered List

  • Apples
  • Bananas
  • Cherries

Ordered List

  1. Wake up
  2. Eat breakfast
  3. Study HTML

Nested List

  • Fruits
    • Apple
    • Banana
    • Orange
  • Vegetables
    • Carrot
    • Potato

Image practice

Use the <img> tag to show an image. Example below:

<img src="tree.jpg" alt="A tall green tree" width="300" height="200">

Video practice

The <video> tag allows embedding videos. Add controls to play, pause, or adjust volume.

<video src="sample.mp4" width="400" controls></video>

Audio practice

The <audio> tag lets you play sound files. Example:

<audio src="sound.mp3" controls></audio>

Practice Section

Try these exercises:

  • Create a new unordered list with at least four of your favorite foods.
  • Make an ordered list showing your daily study schedule.
  • Insert your own image and write a short caption under it using a paragraph.
  • Insert your own short video and add a colorful caption below it.
  • Use the font tag to style some words in your own sentences.

Summary

On this page, you practiced lists, images, videos, audio, and the font tag. Even though some of these features (like the font tag) are old, learning them helps you understand how HTML evolved.
