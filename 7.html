HTML Practice — Links and Anchors

In this lesson, you'll learn how to create links using the <a> tag. Links (also called anchors) connect web pages together and help users navigate through a website or to other pages on the internet.

Basic Link Structure

The basic syntax of a link is:

<a href="URL">Link Text</a>

href stands for "hyperlink reference" — it tells the browser where the link should go.

Example:

<a href="https://www.example.com">Visit Example</a>

Result:

Visit Example

Opening Links in a New Tab

You can make a link open in a new tab using the target attribute with the value _blank.

<a href="https://www.wikipedia.org" target="_blank">Open Wikipedia in new tab</a>

Result:

Open Wikipedia in new tab

Linking to Pages in the Same Folder

If you have another HTML file in the same folder, you can link to it using its filename instead of a full URL.

<a href="about.html">About Us</a>

Result (only works if you have a file named about.html):

About Us

Linking to a Specific Section on a Page

You can jump to a specific part of a page using an id attribute and a link with a # symbol before the id name.

<a href="#contact">Go to Contact Section</a>
...
<h2 id="contact">Contact Us</h2>

Try it below:

Go to the bottom of this page

Email and Phone Links

Use mailto: for email links and tel: for phone numbers.

<a href="mailto:info@example.com">Send an Email</a>
<a href="tel:+1234567890">Call Us</a>

Result:

Send an Email
Call Us

Using Images as Links

You can turn an image into a clickable link by placing the <img> tag inside an <a> tag.

<a href="https://www.example.com">
  <img src="logo.png" alt="Website Logo" width="100">
</a>

Try these exercises:

  • Create a link to your favorite website.
  • Make that link open in a new tab.
  • Add a second link that points to another HTML page in your folder.
  • Add a link that scrolls to a specific section on your page (use an id).
  • Create an email link and a phone number link.
  • Insert an image that acts as a link to another site.

Summary

In this lesson, you learned how to use the <a> tag to create different types of links: external links, internal links, section jumps, and even clickable images. Links are the backbone of the web, connecting pages and information together.
