HTML Practice — More Formatting Tags

This page helps beginners practice a few more inline and visual HTML elements like mark, small, del, strike, and quote. It also shows how to insert an image and write comments inside your code.

Mark tag

The mark tag highlights text as if it was highlighted with a marker pen. Use it to draw attention to important parts of a sentence.

Example: HTML is fun to learn when you practice regularly.

Small text

The small tag displays text in a smaller size, usually for side notes or fine print.

Example: This is a tiny note under the paragraph.

Deleted and Strikethrough text

The deleted tag shows content that has been removed, while the strike tag just visually crosses out text without implying deletion.

Example: The old price was $50 but now it's $35. Or just show something no longer valid: Outdated info.

Quotes

Use quote for short inline quotations inside text, and

for longer quotes that deserve their own block of text.

Inline quote example: My teacher said, Practice makes progress.

Long quote example: HTML gives structure to your content. The more you write, the clearer it becomes that good structure is like good grammar — invisible when done right.

Comments

Comments in HTML are notes that appear in the code but not on the page. They help you remember what each part does.

Example (view it in the source code):

<!-- This is a comment. It won't show up in the browser. -->

Inserting images

Use the <img> tag to insert an image. It needs at least two attributes: src (source of the image file) and alt (a short description for accessibility).

Example:

<img src="flower.jpg" alt="A red flower" width="300">

Practice section

1. Add a sentence where you use the mark, small, and quote tags together.

2. Create your own line with a deleted word and a strikethrough word.

3. Insert an image of your choice (make sure the file is in the same folder). Write a meaningful alt description.

4. Add one HTML comment above your image describing what it shows.

5. Try editing the text to see how each tag changes the look of your content.

Finish line

You’ve now practiced: <mark>, <small>, <del>, <s>, <q>, <blockquote>, <img>, and <!-- comments -->. Save and refresh your file to see how each works.
