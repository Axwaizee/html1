HTML Practice — Working with Tables

In this lesson, you'll learn how to create tables in HTML. Tables help display information in rows and columns, such as schedules, prices, or student lists.

Basic Table Structure

A table starts with the <table> tag and is made up of table rows (<tr>), table headers (<th>), and table data cells (<td>).

Here’s a simple example:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>20</td>
    <td>USA</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>22</td>
    <td>Canada</td>
  </tr>
</table>

Example Output

Name Age Country
Alice 20 USA
Bob 22 Canada

Adding More Rows and Columns

You can add as many rows (<tr>) and columns (<td>) as needed. Every row should have the same number of data cells to keep the table aligned.

Merging Cells with colspan and rowspan

Sometimes you need a cell to stretch across multiple columns or rows. Use colspan and rowspan for that.

Example:

<table border="1">
  <tr>
    <th colspan="2">Merged Header</th>
  </tr>
  <tr>
    <td>Left</td>
    <td>Right</td>
  </tr>
</table>
Merged Header
Left Right
